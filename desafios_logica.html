Treine sua Lógica

1. Ordenar Números

Dado um conjunto de números, descreva os passos para colocá-los em ordem crescente.

Exemplo de solução em pseudocódigo:

para cada posição i na lista
    para cada posição j após i
        se lista[i] > lista[j]
            troque lista[i] com lista[j]

2. Número Primo

Escreva os passos para verificar se um número é primo.

Exemplo de solução em pseudocódigo:

se n < 2 então não é primo
para i de 2 até raiz quadrada de n
    se n dividido por i for zero
        não é primo
se nenhum divisor foi encontrado
    então é primo

3. Palíndromo

Descreva uma lógica para verificar se uma palavra ou frase é um palíndromo (lê-se igual de trás para frente).

Exemplo de solução em pseudocódigo:

remova espaços e coloque tudo em minúsculas
crie uma string invertida
compare a original sem espaços com a invertida
se forem iguais, é palíndromo
